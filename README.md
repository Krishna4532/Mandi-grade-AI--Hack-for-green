# 🌾 Mandi Grade AI
**Real-time Agricultural Quality Assessment & Price Valuation**

Mandi Grade AI is a professional-grade solution designed to empower farmers and traders. By leveraging AI-driven visual analysis, it provides instant quality grading and fair market valuation, ensuring transparency in the agricultural supply chain.

### ✨ Key Features
* **AI Vision Grading:** Instant classification of crops (Premium, FAQ, Under-Grade).
* **Professional Reporting:** Generates a high-fidelity "Quality Report Card" for every scan.
* **Live Market Sync:** Simulates a real-time connection to Mandi price feeds.
* **Interactive UI:** High-end glassmorphism design with agricultural themes and scanning animations.

### 🚀 Quick Start
1. Clone the repository: `git clone <your-repo-link>`
2. Install dependencies: `pip install flask`
3. Launch the portal: `python app.py`
